# 如何啟動專案

以下是啟動專案的步驟：

1. 克隆此存儲庫到本地機器：

    ```
    git clone https://github.com/your_username/project_name.git
    ```

2. 進入專案目錄：

    ```
    cd project_name
    ```

3. 安裝相關依賴：

    ```
    npm install
    ```

4. 啟動專案：

    ```
    npm start
    ```
